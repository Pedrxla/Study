      Laço while no JavaScript

        Neste exemplo, o while adiciona itens à lista até que o contador i atinja o valor 5. O loop vai executando enquanto a condição i <= 5 for verdadeira. Diferença prática entre for e while: for: Útil quando o número de iterações é conhecido ou baseado em uma sequência (ex: for (let i = 0; i < 10; i++)). while: Ideal quando você não sabe o número exato de iterações e precisa repetir até que uma condição seja satisfeita.
